## Getting Component Code

1. Visit the component page at https://www.ui-layouts.com/components/[component-name]
2. Copy the component source code from the code tab
3. Paste into your project
4. Install any extra dependencies shown in the component's installation section

## Theming

UI-Layouts uses CSS variables for light/dark mode. Add these to your global CSS:

```css
:root {
  --background: 0 0% 100%;
  --foreground: 222.2 84% 4.9%;
  --border: 214.3 31.8% 91.4%;
  --primary: 222.2 47.4% 11.2%;
  --muted: 210 40% 96.1%;
  --radius: 0.5rem;
}

.dark {
  --background: 228 86% 5%;
  --foreground: 210 40% 98%;
  --border: 217.2 32.6% 17.5%;
  --primary: 210 40% 98%;
  --muted: 217.2 32.6% 17.5%;
  --radius: 0.5rem;
}
```
